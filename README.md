# Phishing Simulation – Internee.pk

Phishing awareness project using **GoPhish** to simulate phishing attacks against Internee.pk’s team. The simulation tested employee awareness, tracked interactions, and generated a security training report to strengthen defenses against social engineering.

## Summary of Security Awareness Findings

- Employees interacted with phishing emails (opened, clicked links).
- Landing page visits confirmed susceptibility to phishing attempts.
- Awareness training required to reduce risk of real-world phishing.

## Results & Learnings

### Phishing Simulation Skills

- Configured GoPhish campaigns with SMTP integration.
- Designed phishing templates and landing pages.
- Monitored campaign analytics and user interactions.

### Security Awareness Practices

- Reinforced importance of **verifying email sources**.
- Highlighted risks of clicking suspicious links.
- Demonstrated need for **continuous employee training**.

### Practical Skills Gained

- GoPhish installation and configuration.
- Mailtrap sandbox integration for safe testing.
- Campaign design, execution, and reporting.
- Documentation of phishing awareness results.
